# Frontend
From root folder  :
* `cd conversion_app_client`
* `npm i`
### run tests
* For unit test : `npm run test`
### run app 
* To run application : `npm run start`

Then,
Go to `http://localhost:4200`


# Backend
From root folder:
* `cd conversion_app_server`
* `mvn install
* Set your SDK to Java 8
* Run the Application

### run tests
* junit tests

Then,
Go to `http://localhost:8056/unit-conversion`
